### 目录结构介绍
+ pool 
	翻译池中存放着准备翻译的文章，任何人可以pull request到pool中添加优秀的英文文章
+ doing
	为正在翻译的文章
+ reviewing
	为正在被review的文章
+ done
	为review完成准备发布的文章
+ join
	为想要加入翻译小组的成员，首先要从pool中，选区一篇文章进行翻译，然后提交到join中，通过review后，加入到members中去
+ members
	为前端翻译小站成员
